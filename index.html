Basic Installation

Hello! Welcome to official documentation. Braine - Braine - Startup Portfolio Website Template is a perfect solution for creatng a professional looking portfolio website for your startup, agency or any other business. It is build with Next JS, React JS, Tailwind CSS. Lets get started.

Prerequisites

React is built using Node.js, so you will need to have latest version of Node.js installed on your machine. NPM automatically gets installed with Node.js.

Installing Development Environment

Move to /braine-startup-template (project root) in your terminal/comand prompt. You can aslo open the project root folder in your vs code editor and open terminal in vs code. Whichever you prefer. Now run the following command in your terminal/vs code terminal.

npm install

This will download and install all the necessary packages listed in the package.json file. After Installation you should have a /node_modules folder in your project root. Ensure your internet connection is stable during this process.

How To Run Development Server

Once dependencies are installed, start the development server with the following command:

npm run dev

The server will start, and you should see the URL for accessing the project locally in your terminal (e.g., http://localhost:3000). Open the url in your favorite browser and you should see a page like this.

Any changes you make to the code will automatically reflect in the browser, thanks to hot module reloading.

Env File Setup

Create a file named .env.local / .env and copy the contents of the .env.example file into it. We will talk about how to get your email js credentials used in the env file in the email js setup section.

Build for Production (Optional)

When you are ready to deploy the website, generate an optimized production build by running:

npm run build

After the build completes, you can preview the production build locally using:

npm run start

This command will start a server running the optimized production build. You can now deploy the website to your favorite hosting service if you want.

Troubleshooting

If you encounter any issues during installation or setup, ensure the following:

  • Your Node.js version meets the project's requirements (usually the latest LTS version).
  • You have a stable internet connection for downloading dependencies.
  • No other application is running on localhost:3000.

For additional support, you can refer to the Next.js documentation or contact us through the support system on ThemeForest.

Congratulations!

You have successfully set up the Braine - Startup Portfolio Website Template. Enjoy building your professional portfolio!

EmailJS Setup (For Contact Us Form)

We will set up email js in this step and will use the credentials found in the env file to make the contact us form work.

Step-1: Create EmailJS Account

Go to EmailJS website and create an account using sign up button. Login to your account and you will see a dashboard like below. (You might not have any service added at first)

Step-2: Add A Service

If you already have a service added, you can click the edit button and skip Step-2. Otherwise, click on the Add New Click on the Add New Service button. You will see a window like below. Select You Desired Option.

Step-3: Add A Service

In both cases (Edit or add service), you will see a window like below. Connect Your Account using the connect button. Then copy the service id and paste it in the NEXT_PUBLIC_EMAILJS_SERVICE_ID field in your env file. Then click create service button to end. (see image below)

Step-4: Get Your Template Id

Click on the template tab from the left sidebar, you will see a default template like below. Copy the templated id info from it and paste it in the NEXT_PUBLIC_EMAILJS_TEMPLATE_ID field in your env file. (see the image below)

Step-5: Get Your Public Key

Click on the account tab from the left sidebar, you will see a window like below. Copy the public key info from it and paste it in the NEXT_PUBLIC_EMAILJS_PUBLIC_KEY field in your env file. (see the image below)

Step-6: Edit Your Template (Optional)

You use the default template. If you want you can edit it to your liking by clicking o edit icon.

If you face any difficulty, you can contact us through the support system on ThemeForest.

Customization

Color Palette Customization

Styling is done using Tailwind CSS. These are are the majors colors used in the template. You can find them in tailwind.config.ts file. You can customize them to your liking. There is also a dark mode option. You can also change the color of the dark mode to your liking. To change the dark color you have to replace colors used in the dark: classes in the html elements/tags. You can change non-majors colors directly in the element where they are used.

Typography & Base Styles Customization

Lexend is used as font in this website. You can import another font from google fonts and use it. You can also change the base styles used for this template by going to /src/app/global.css file.

Logo Customization

You can also customize the logo text by changing text of Brand portion's Link tag. You can also place a logo image of your company with logo text inside the Link tag. You can find this portion of code in the /src/components/shared/Navbar.tsx file.

Navbar Customization

You can customize the navbar by going to /src/components/shared/Navbar.tsx file. To change a navbar item you have the change the text between the Link tag eg: About(as seen in the image). You can change the url/link in the href attribute of the Link tag. If you change the link/url in href attribute make sure to update their ids in their corresponding components. For Example if you change the about url/link of About, you have to update id of it in the about component (AboutUS.tsx) to ensure on click it takes to the correct section of the website/page.

Apart from this, you can also change the url of LinkedIn button of the navbar to your company's LinkedIn page.

Footer Customization

Like the navbar, you can customize the brand text / logo in the footer. You can link your social media account using the icon's link tags. You can also customize the copyright text and the year in the footer. You can replace the contact information with your own contact information. And of course you can link additional web pages here your privacy policy, terms and conditions, etc. (by changing the href attribute of the link tag). You can customize the footer by going to /src/components/sections/Footer.tsx file.

Homepage Customization

You can customize the order of the section in the template from this page. If you want to add any extra section you can also do it here. You can customize the footer by going to /src/app/(with-navbar)/page.tsx file.

Banner Section Customization

You can customize the typewriting text of the banner by changing the words of values attribute in the TypeWriter tag (as seen in the image). You can also change other texts of the banner. You can customize the banner by going to /src/components/sections/Banner.tsx file.

You can change the video of banner by changing the src of the video and replacing it with the video you want. Just make sure to place the video file in /public/videos folder of the project. You can also change the background image of the banner by changing the src attribute of the image tag. You can can customize these by going to /src/components/VideoImage.tsx file.

Services Section Customization

If you want to update the services cards of this template, you can do it by replacing the animation, title, and description attribute value of the Service tag/component. (as seen in the image)> You can also add more services by adding/calling Service tag/component with the above mentioned attributes. You can customize the services section by going to /src/components/sections/Services.tsx file.

You can also customize the Title, Heading and Sub Heading of the Services section by changing the values of titleText, headingText and subHeadingText attributes of the SectionHeader tag/component. (as seen in the image).

You can find the animations in Lottie Files Animation.

About Us Section Customization

You can customized the images by changing the src attribute of the image tag. You can also change the texts of About Us, Mission, Vision by updating their texts inside the getTabContent() function of the AboutUs.tsx file. You will find the About Us section in the /src/components/sections/AboutUs.tsx file.

You can also customize the Title, Heading and Sub Heading of the About Us section by changing the values of titleText, headingText and subHeadingText attributes of the SectionHeader tag/component.

Our Team Section Customization

If you want to update the team card or the information of the team member of this template, you can do it by replacing the image, name, role, email, linkedinLink and githubLink attribute value of the TeamMember tag/component with your own values. (as seen in the image) You can also add more team members by adding/calling TeamMember tag/component with the above mentioned attributes. You can customize the Our Team section by going to /src/components/sections/OurTeam.tsx file.

You can also customize the Title, Heading and Sub Heading of the Our Team section by changing the values of titleText, headingText and subHeadingText attributes of the SectionHeader tag/component. (as seen in the image).

Portfolio Section Customization

In the portfolio section you can change the projects of each category by going to their respective component like All.tsx, BrandingStrategy.tsx etc. If you want to update a specific project of a category, you can do it by updating name, image src, category text. (as seen in the image). You can also change category names if you want. You can customize the Portfolio section by going to /src/components/sections/Portfolio.tsx file and then going to category components.

Testimonials Section Customization

If you want to update the testimonial card or the information of a testimonial of this template, you can do it by replacing the feedbackText, name, role, and img attribute value of the Testimonial tag/component with your own values. (as seen in the image) You can also add more testimonials by adding/calling Testimonial tag/component with the above mentioned attributes. You can customize the Testimonials section by going to /src/components/sections/Testimonials.tsx file.

You can also customize the Title, Heading and Sub Heading of the Testimonials section by changing the values of titleText, headingText and subHeadingText attributes of the SectionHeader tag/component. (as seen in the image).

Trusted By Section Customization

You can change the brands of the Trusted by section by replacing the current brand images with the new brand images. You can customize the Trusted By section by going to /src/components/sections/TrustedBy.tsx file.

Blogs Section Customization

If you want to change the blog secton of this template you can update in the /data/data.json file. (as seen in the image) You can customize the blog images and other texts in the /src/components/sections/Blogs.tsx file.

Support Section Customization

You can customize support email, office location and other links by going to /src/components/sections/Support.tsx file.
